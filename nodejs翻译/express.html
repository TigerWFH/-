Express API 翻译

Request:

request对象封装了http请求的所有信息。该对象的属性有request query string,parameters,body,HTTP headers等等。

在本文档中，为了方便，会简写为req（响应简写为res），但是他的实际命名是根据你的回调函数中的参数指定的。例如：

app.get('/user/:id', function(req, res){
res.send('user' + req.params.id);
});

或者

app.get('/user/:id', function(request, response){
res.send('user' + request.params.id);
});

Properties

在版本4中，默认的req.files移除，如果要访问上传的文件对象，请使用有关插件。multer便是其中之一。

req.app
该属性存储一个正在应用模块的express应用实例的引用。按照指定的方式(使用exports导出模块，使用require加载模块)书写插件，模块通过该属性可以与express应用交互。例如：

//index.js
app.get('/viewdirectory', require('./mymiddleware.js'));

// mymiddleware.js
module.exports = function (req, res){
res.send('The views directory is' + req.app.get('views'));
};

req.baseUrl
路由实例挂载的url路径。req.baseUrl属性和app应用的mountpath属性类似。app.mountpath会返回匹配的所有路径模式。例如：

var greet = express.Router();
greet.get('/jp', function(req, res){
console.log(req.baseUrl); // /greet
res.send('Konichiwa!');
});
app.use('/greet', greet); //load the router on '/greet'
